\subsection{Detailed Description}
Allows debug ouput and assert usage. Usage\-: Before including this file, define D\-E\-B\-U\-G as the debuglevel, eg\-:


\begin{DoxyCode}
\textcolor{preprocessor}{#define DEBUG 1}
\end{DoxyCode}


for debuglevel 1. Then use debug\-Print(\char`\"{}\-Foo\char`\"{}) in your code. If you need to calculate stuff for your debug output, enclose it\-:


\begin{DoxyCode}
\textcolor{preprocessor}{#if DEBUG >= 1}
\textcolor{preprocessor}{}    \hyperlink{group__debug_gada1787bea8c10cd89775d6918f6fd324}{debugPrint}(\textcolor{stringliteral}{"Bar"});
\textcolor{preprocessor}{#endif}
\end{DoxyCode}

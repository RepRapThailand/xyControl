3 [xyControl][gh] is a Quadrocopter Flight Controller based on Atmels Atmega2560 microcontroller.
4 It features 512KB SRAM on-board, using the external memory interface of this processor.
5 Also included is a switched power supply as well as a USB connection to communicate with and program the target.
6 All I/O pins, including 3 additional UARTs, SPI, I2C (TWI) and 16 ADC Channels, are accessible via standard 2.54mm connectors.
7 The Board can be powered from an external stable 5V supply, USB or 7V or more, via the on-board switched power supply. All voltage sources can be selected via jumpers.
